Comparison of the performance of Indonesian Tala and Nazhief Adriani stemming algorithms using python

Methodologies

In this research, there are 3 main steps including scraping, generating term and stemming

  1. Scraping documents
    In this stage, 50 documents from the rumahginjal.id site are taken.
  2. Generating Term
    This stage consists of several steps, they are:
    1. Remove punctuation contained in the document.
    2. Removes excess space contained in the document.
    3. Separating terms based on spaces, where the terms in the whole document will be separated based on spaces. Because spaces are the link between each word in a sentence.
    4. The letters in each term will be changed to the lowercase form.
    5. Add trim terms to the database and calculate the frequency of each term.
  3. Stemming
    This stage is carried out the stemming process for each term using the Tala and Nazief & Adriani algorithm while calculating the processing time and checking the validity of the stemming results from each algorithm
